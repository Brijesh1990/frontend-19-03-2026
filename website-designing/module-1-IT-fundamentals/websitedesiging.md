# what is website designing ?
  1. website designing is collection of webpages 
  2. website designing is create Home page | webpages
  3. website designing create GUI | UI 
  4. website designing using HTMl | CSS | javascript
  5. website designing create landing page or Home page

  **examples of website designing in html structures** 

  ```
<!DOCTYPE html>
<html>
<head>
<meta charset='utf-8'>
<meta http-equiv='X-UA-Compatible' content='IE=edge'>
<title>Page Title</title>
<meta name='viewport' content='width=device-width, initial-scale=1'>
<link rel='stylesheet' type='text/css' media='screen' href='main.css'>
<script src='main.js'></script>
</head>
<body>
<!-- display in webpage -->
</body>
</html>

  ```
