Filter Shadow

The filter-shadow property is used to apply a shadow effect to an element. It allows you to create a drop shadow that can be customized with various parameters such as color, offset, and blur radius.

In the example above, we have applied two drop shadows to the image. The first shadow is red with an offset of 10px in both the x and y directions and a blur radius of 5px. The second shadow is blue with an offset of -10px in both the x and y directions and a blur radius of 5px. This creates a layered shadow effect around the image.
